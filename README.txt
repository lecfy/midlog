Midlog - простий блог для публікації ваших думок
Вимоги: php7, mysql pdo, rewrite

Встановлення
1) Розпакувати архів
2) Створити базу даних та користувача з правами до неї
3) Відредагувати файл app/Config.php
4) Імпортувати midlog.sql в базу даних

Done:) Перший зареєстрований користувач буде адміністратором блогу
